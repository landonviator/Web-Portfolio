Viator Pre-Mix

The Pre-Mix plugin is a multi-purpose, first-instance plugin that combines some of the most common preliminary audio mixing techniques needed to get an audio source ready for the mixing stage. The Pre-Mix contains three basic audio circuits: Input Gain, Filter, Saturation.
Input Gain

The input stage has a few handy parameters. Firstly, there's a slider to change the gain. But, operating a slider or knob for fine-tuning can be unwieldy, so there are a pair of buttons to add increments of +1.5 or -1.5 dB of gain. The "fine" toggle changes these buttons to +.25 and -.25 dB. Double clickng the slider or clicking the "unity" button will bring the slider to 0 db.

Filter

The filter section is straight-forward, there's a low-pass and high-pass filter. Although intended to cut the very top and bottom of the frequency range, the knobs have a full 20-20kHz range to suit any sound source.

Saturation

Here's where the fun begins! The goal of this circuit is to emulate an analog console or preamp. The saturation circuit drives the input and can be used as a subtle overdrive to and insane distortion. The "mode" menu allows you to choose from four distortion modes: subtle, soft, driven, insane.

Each is an incresingly heavier level of distortion. Turning the input gain will drive the signal into the saturation harder to achieve even more distortion, or you can pull the input down to get an even more subtle level of distortion. The voice option allows you to choose either a "vintage" or "modern" voiceing. The vintage voice will add high-end and low-end shelfs, increasing in dB amount as the saturation goes up; the more saturation, the more high and low end get bumped. The modern setting does the same thing with the mid range instead. If you want a clean frequency response with no eq bump, you can simply select "voice" again to clear it. Lastly, there is an output slider to adjust the volume with an "auto" knob that dynamically adjusts the volume as you adjust the distortion. Since the different modes add more distortion, the auto function will automatically adjust and control the volume accordingly.
