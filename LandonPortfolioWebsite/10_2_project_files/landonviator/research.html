Hindmarsh-Rose



The Hindmarsh-Rose neural model is a continuous-time chaotic system. see James L Hindmarsh and RM Rose, “A model of neuronal bursting using three coupled irst order differential equations,”Proceedings of the Royal society of London. Series B. Biological sciences221, no.1222 (1984): 87–102.

It was first described by Hindmarsh and Rose in 1982. This model is an adaptation of a previously described model by Fitzhugh. This model described the behavior of a brain cell discovered in a pond snail that when depolarized by a short pulse of current, an action potential (burst) was created with a short after potential decay; the cell does not fire indefinitely. To produce the behavior of the cell, the introduction of a small electrical current is used. A slow, increasing current causes adaptations in the cell, so the Hindmarsh-Rose model adds an adaptation current to the original equation. The Hindmarsh-Rose model is described by the following differential equations:

\[\dot{x} = y - ax^3 + bx^2 + I -z\] \[\dot{y} = c - dx^2 - y\] \[\dot{z} = r(s(x - x_1) - z)\]


The image above is an x-y, oscilloscope rendering of the Hindmarsh-Rose system that I implemented in MaxMSP. The Hindmarsh-Rose chaotic equations were implemented in the gen environment in MaxMSP by finding numerical solutions to the above equations. Included is an audio sample of the oscillator with a specific set of conditions, but the possible timbrel range of this oscillator is infinitely large.

Chaotic String

The chaotic string is a digital waveguide that uses the Hindmarsh-Rose oscillator as the noise source for the envelope. Injecting the chaotic oscillator inside of this synthesis engine in a modular way avoids the pit falls usually experienced by chaotic sound synthesis, like bifurcation or the system failing altogether when changing parameters. In this implementation, the oscillator is set at a stable value then used as a norse burst for the string model.

The result of using this chaotic oscillator as the noise source for the digital waveguide is an interesting string sound sound that is ambient in nature and rich in high-end frequencies and harmonic saturation. It is a much more novel string sound than the standard digital waveguide sound.
